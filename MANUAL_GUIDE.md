## 1. 商品を非表示にする（ブロック機能）

表示したくない商品を、ボタン1つで非表示にできます。

### 手順

**ステップ1：管理モードでサイトを開く**

URLの末尾に `?admin=1` を追加してブラウザで開きます。

```
例: https://honestbaby-care.com/?admin=1
```

> ※ 一度管理モードで開くと、ブラウザを閉じても次回から自動的に管理モードになります。

**ステップ2：非表示にしたい商品を見つける**

商品カードの左上に赤い「×」ボタンが表示されています。

**ステップ3：「×」ボタンを押す**

押すと即座に商品が消えます。この操作はデータベースに保存されるので、次回サイトを開いても非表示のままになります。

## 2. ブロックを解除する

非表示にした商品を元に戻すには、Supabaseのダッシュボードから操作します。

### 手順

**ステップ1：Supabaseにログイン**

[https://supabase.com](https://supabase.com) にアクセスし、プロジェクトにログインします。

**ステップ2：テーブルを開く**

左のメニューから「Table Editor」を選び、`product_blocklist` テーブルを開きます。

**ステップ3：行を削除する**

解除したい商品の行を選択し、削除ボタン（ゴミ箱マーク）を押します。

> ※ 削除後、サイトをリロードすると商品が再び表示されます。

## 3. 商品を手動で追加する

楽天・Yahoo以外の商品を手動でデータベースに登録できます。

### 手順

**ステップ1：Supabaseにログイン**

[https://supabase.com](https://supabase.com) にアクセスし、プロジェクトにログインします。

**ステップ2：productsテーブルを開く**

左のメニューから「Table Editor」を選び、`products` テーブルを開きます。

**ステップ3：「Insert row」をクリック**

右上の「Insert row」ボタンを押して、新しい行を追加します。

**ステップ4：情報を入力する**

| フィールド名 | 内容 | 例 |
|------------|------|-----|
| `name` | 商品名 | エルゴノミック 抱っこ紐 |
| `price` | 価格（円） | 15800 |
| `rating` | 評価（4.0〜5.0） | 4.7 |
| `reviews_count` | レビュー数 | 120 |
| `image_url` | 商品画像のURL | https://... |
| `category` | カテゴリ名（サイトのカテゴリと一致させる） | 抱っこ紐 |
| `sub_category` | サブカテゴリ名 | 縦抱き |

**ステップ5：「Save」で保存**

「Save」ボタンを押して保存します。サイトをリロードすると商品が表示されます。

## 4. カテゴリ・サブカテゴリを変更する

カテゴリやサブカテゴリの名前・内容を変えるには、コードを直接編集します。

### 対象ファイル

```
/src/App.jsx  ← このファイルの上部にある CATEGORY_TREE を編集する
```

### CATEGORY_TREE の構造

```javascript
const CATEGORY_TREE = [
  {
    name: "おむつ",          // カテゴリ名（サイトに表示される名前）
    id: "205197",            // 楽天のジャンルID（変更不要）
    keyword: "おむつ",       // 楽天検索に使うキーワード
    icon: "🩲",             // アイコン（絵文字）
    subs: [                  // サブカテゴリの一覧
      {
        name: "テープタイプ",                                          // サブカテゴリ名
        subsubs: ["新生児", "S", "M", "L", "BIG", "BIGより大きい"]    // 3段階目（サイズなど）
      },
      { name: "パンツタイプ", subsubs: ["S", "M", "L", "BIG", "BIGより大きい"] },
      { name: "おしりふき" },  // subsubs不要なら省略OK
    ]
  },
  {
    name: "ベビーカー",
    id: "200833",
    keyword: "ベビーカー",
    icon: "🚼",
    subs: ["A型", "B型", "AB型", "バギー", "周辺グッズ"]  // シンプルな文字列配列もOK
  },
  // ...
];
```

### サブカテゴリを追加・変更する

`subs` の配列に文字列を追加・変更するだけです。

```javascript
// 変更前
subs: ["A型", "B型", "AB型", "バギー"]

// 変更後（「三輪タイプ」を追加）
subs: ["A型", "B型", "AB型", "バギー", "三輪タイプ"]
```

### 3段階目（サイズ等）を追加する

サブカテゴリをオブジェクト形式にして `subsubs` を追加します。

```javascript
// 変更前（文字列）
subs: ["テープタイプ", "パンツタイプ"]

// 変更後（3段階目を追加）
subs: [
  { name: "テープタイプ", subsubs: ["新生児", "S", "M", "L"] },
  { name: "パンツタイプ", subsubs: ["S", "M", "L"] },
]
```

### 注意事項

- `id`（楽天ジャンルID）を変えると検索結果が変わります。正しいIDは楽天の開発者ページで確認できます。
- カテゴリ名を変更した場合、Supabase の `products` テーブルの `category` 列も合わせて変更が必要です。

## 管理モードの解除方法

管理モードを解除したい場合は、ブラウザの開発者ツールを使ってlocalStorageを削除します。

**Chrome/Edgeの場合：**
1. F12キーで開発者ツールを開く
2. 「Application」タブ → 「Local Storage」→ サイトのURLを選ぶ
3. `honestBabyAdmin` の行を削除する
4. ページをリロードする
